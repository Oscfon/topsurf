<h1><p align="center">topsurf</p></h1>

toposurf is a Python package for working with cell decompositions of surfaces
in [SageMath](https://sagemath.org).

sage-flatsurf is free software, released under the [GPL v2 (or later)](./COPYING).

We welcome any help to improve topsurf. If you would like to help, have
ideas for improvements, or if you need any assistance in using topsurf,
please don't hesitate to [contact us](https://flatsurf.github.io#contact).
